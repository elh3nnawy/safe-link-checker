# Safe Link Checker A Python Flask web application that checks if a URL is safe using the Google Safe Browsing API. It flags malicious links in real time and demonstrates cybersecurity awareness, API integration, and web development skills.
## Features 
-Real-time URL safety check 
-Integration with Google Safe Browsing API 
-Simple Flask web interface
-Secure API key handling with `.env`
-Easy to extend with logging or ML
## 🛠️ Tech Stack
- Python 3
- Flask
- Requests
- Jinja2
- Google Safe Browsing API
## ⚙️ Setup Instructions 
1. **Clone the repo** ```bash git clone https://github.com/elh3nnawy/safe-link-checker.git cd safe-link-checker
2. **Install dependencies** --- "pip install -r requirements.txt"
3. **Add your Google API key to .env**
4. **Run the app** -- "python app.py"
